# AI 語音情緒陪伴助手 - 後端

這是一個基於 FastAPI 的輕量級後端伺服器，完全不依賴任何本機端的龐大 AI 模型。
所有的語音辨識、圖片情緒分析與對話生成，都是透過呼叫 **Google Gemini API** (`gemini-2.5-flash`) 雲端完成。

## 快速啟動指南 (本機執行)

### 第一步：安裝 Python 套件

請打開終端機 (Terminal 或 PowerShell)，並確認目前路徑在 `backend` 資料夾下。
因為移除了 DeepFace，現在的套件非常輕量，請直接執行：

`pip install -r requirements.txt`

*(目前必要的套件只有 `fastapi`, `uvicorn`, `python-multipart`, 與 `google-generativeai`)*

### 第二步：設定環境變數

在 `backend` 目錄下建立一個 `.env` 檔案（如果已經有就不需要重建），填入您的 API Key：

`GEMINI_API_KEY=your_gemini_api_key_here`

### 第三步：啟動伺服器

執行以下指令啟動 FastAPI 後端伺服器：

`uvicorn main:app --reload --port 8000`

可內網連接請用

uvicorn main:app --host 0.0.0.0 --port 8000 --reload

啟動成功後，您會看到 `Uvicorn running on http://0.0.0.0:8000` 的綠色成功訊息。
（由於移除了 TensorFlow，現在啟動應該只要一瞬間，且不會再有任何紅字警告！）

### 第四步：測試是否成功

您可以開啟瀏覽器，輸入 `http://localhost:8000/config`，如果看到以下畫面，代表設定大成功：

`{"gemini_ready":true}`
