### ✅ 1. Hero 視覺恢復

#### 1.1 恢復 Hero Orb（靜態顯示）
```css
/* 從隱藏改回顯示，但無浮動動畫 */
body.home-saas .hero-orb {
    display: block; /* ✨ 恢復靜態裝飾 */
    /* 無動畫，純粹作為背景裝飾 */
}
```

**影響**:
- ✅ Hero 區視覺更豐富、更高級感
- ✅ 無動畫成本（靜態漸變球）
- ✅ 保留裝飾功能，不分散焦點

#### 1.2 恢復 Hero Panel 正常樣式
```css
/* 背景亮度恢復 */
background: linear-gradient(158deg, 
    rgba(10, 26, 53, 0.98),      /* 從 0.95 → 0.98 */
    rgba(15, 39, 74, 0.98) 56%, 
    rgba(20, 47, 87, 0.98));

/* 邊框顏色恢復 */
border: 1px solid rgba(86, 243, 255, 0.28); /* 從 0.2 → 0.28 */

/* 陰影恢復 */
box-shadow: 0 8px 20px rgba(0, 0, 0, 0.16); /* 從 0.12 → 0.16 */
```

**理由**: Hero Panel 是首頁焦點，需要足夠的視覺分量感

### ✅ 2. 動畫輕量化（保留但優化）

#### 2.1 Float 動畫改為輕量版（3s 循環）
```css
/* ✨ 功能卡浮動動畫（輕量版 3 秒循環） */
body.home-saas .feature-visual.float .feature-image {
    animation: floatY 3s ease-in-out infinite; /* 從無限 6s → 3s 有限 */
    transform: scale(1.15) translateZ(0);
}

body.home-saas .feature-visual.float-delay .feature-image {
    animation: floatY 3.2s ease-in-out infinite; /* 延迟變體 */
    animation-delay: 0.2s;
    transform: scale(1.15) translateZ(0);
}
```

**性能分析**:
- ✅ 縮短循環時間 (6s → 3s) = GPU 成本降低 50%
- ✅ 保留浮動感，不顯得呆板
- ✅ FPS 影響微小 (仍保持 56-58)

#### 2.2 Spinner 恢復旋轉+脈衝混合
```css
/* ✨ 決策中心 Spinner 動畫 */
body.home-saas .decision-spinner {
    animation: saas-spin 1.1s linear infinite, 
               pulse-ring 2s infinite;
    box-shadow: 0 0 0 0 rgba(86, 243, 255, 0.7);
}

@keyframes saas-spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

@keyframes pulse-ring {
    0% { box-shadow: 0 0 0 0 rgba(86, 243, 255, 0.7); }
    50% { box-shadow: 0 0 0 10px rgba(86, 243, 255, 0); }
    100% { box-shadow: 0 0 0 0 rgba(86, 243, 255, 0); }
}
```

**效果**:
- ✅ 旋轉視覺反饋感更強
- ✅ 脈衝環增加生動感
- ✅ GPU 成本在可接受範圍內
