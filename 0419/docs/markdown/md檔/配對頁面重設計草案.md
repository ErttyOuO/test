# 配對頁面重設計草案（含淺色/深色模式）

## 1. 目標與設計原則
- 目標一：讓「配對 -> 選擇聯繫 -> 進入對話」流程更直覺。
- 目標二：在桌機與手機都保持清楚層級，不再出現主要內容視覺過空。
- 目標三：完整支援淺色/深色模式，避免只改背景但對比不足。
- 原則：資訊分區明確、主要動作突出、色彩 token 化、動畫有意義。

## 2. 新版資訊架構（IA）
- 區塊 A：左側導覽與收藏（固定寬度）。
- 區塊 B：中央主舞台（配對卡片 / 空狀態 / 聯繫挑選 Modal）。
- 區塊 C：右側上下文欄（可選，桌機顯示，手機收合）。
- 優先流程：
  1. 使用者滑卡配對。
  2. 滑完後跳出「可聯繫保險員」中央 Modal（並排卡片）。
  3. 可選一對一聯繫或改走多人聊天。

## 3. 版面草圖（Desktop）

```
┌──────────────── Header ────────────────┐
│ Logo | Nav | Theme Toggle | User       │
└────────────────────────────────────────┘
┌──────────┬────────────────────────┬──────────┐
│ 左側欄   │ 中央主舞台             │ 右側欄   │
│ - 導覽   │ - 配對卡片堆           │ - 提示   │
│ - 收藏   │ - 動作按鈕             │ - 說明   │
│ - 快捷   │ - 空狀態/完成狀態      │ - FAQ    │
└──────────┴────────────────────────┴──────────┘
```

- 建議寬度：
  - 左側欄：280px
  - 中央主舞台：minmax(640px, 1fr)
  - 右側欄：320px（可選）

## 4. 版面草圖（Mobile）
- Header 保留。
- 左側欄改為抽屜式（Drawer）。
- 主舞台全寬，卡片高度降低 10%~15%。
- 配對完成 Modal 改為單欄卡片 + 底部吸附 CTA。

## 5. 核心元件重設計
### 5.1 配對卡片
- 提升內容可讀性：名字、公司、簡介改為 3 層階。
- 底部動作按鈕改為「語義 + icon」：
  - 略過
  - 喜歡
- 卡片動畫：滑動時只做 transform + opacity，避免 layout reflow。

### 5.2 收藏列表
- 每筆收藏統一 56px avatar + 2 行資訊 + 1 個主按鈕。
- 「可聯繫 / 未聯繫」狀態用 badge，不只靠顏色。

### 5.3 配對完成 Modal（重點）
- 置中大視窗（Desktop 約 860px）。
- 2 欄並排卡片（Mobile 1 欄）。
- 底部文案：
  - 「還是想試試多人聊天？」（可點文字，直接導流）。

## 6. 淺色/深色模式規劃

## 6.1 Theme 切換策略
- 採用 `data-theme="light|dark"` 掛在 `html` 或 `body`。
- 初始值：
  - 優先讀取 localStorage `theme`。
  - 無設定則使用 `prefers-color-scheme`。

## 6.2 設計 Token（CSS Variables）
```css
:root[data-theme="light"] {
  --bg-canvas: #f3f6fb;
  --bg-surface: #ffffff;
  --bg-elevated: #f8fbff;
  --text-primary: #0f172a;
  --text-secondary: #475569;
  --border-subtle: #dbe4ef;
  --accent: #38bdf8;
  --accent-strong: #0284c7;
  --success: #22c55e;
  --danger: #ef4444;
  --shadow-lg: 0 12px 30px rgba(2, 8, 23, 0.10);
}

:root[data-theme="dark"] {
  --bg-canvas: #0b1220;
  --bg-surface: #111a2e;
  --bg-elevated: #16223a;
  --text-primary: #e2e8f0;
  --text-secondary: #94a3b8;
  --border-subtle: #24324a;
  --accent: #38bdf8;
  --accent-strong: #7dd3fc;
  --success: #22c55e;
  --danger: #f87171;
  --shadow-lg: 0 12px 30px rgba(0, 0, 0, 0.45);
}
```

## 6.3 對比規範
- 內文字級 14px 以上，對比至少 4.5:1。
- 大標題與按鈕文字對比至少 3:1。
- 深色模式下避免純黑背景，保留層次（`--bg-canvas` vs `--bg-surface`）。

## 7. 動效與互動
- 卡片進場：stagger 60ms。
- Modal 開啟：scale(0.98->1) + fade 180ms。
- Hover 只在桌機啟用，手機改為 active feedback。
- 尊重 `prefers-reduced-motion`：改為無位移、只淡入。
