## 5. 實作步驟（執行版）

1. 建立子資料夾結構（功能模組/配對、保險後台、點讚留言、共用資產、伺服端_點讚留言）。
2. 依第 3 節將來源檔案先複製到對應子資料夾，不覆蓋主檔。
3. 對照目標根目錄同名檔，逐段整合：
   - HTML：合併缺失區塊與必要掛載點。
   - JS：只搬函式與事件綁定，避免整檔覆蓋。
   - CSS：抽出新增 class，避免覆寫整體主題。
4. 若有後端模型，先在功能模組資料夾內做相容調整，再整併到正式 server。
5. 驗證流程：
   - 配對頁流程
   - 後台登入與後台頁
   - 點讚、留言、貼圖互動
6. 全面驗證通過後，清理目標資料夾內重複舊檔（例如已被整合、且保留於功能模組比對版中的舊頁面或舊腳本），來源資料夾不刪除。
